# 📊 Flipkart Sales Dashboard – Excel Project

Welcome to the **Flipkart Sales Dashboard Project**! 🚀 This repository highlights how raw e-commerce sales data can be transformed into meaningful insights using **Microsoft Excel**. The dashboard is designed to be interactive, visually appealing, and business-friendly.

---

## 🛍️ Project Overview

This project analyzes Flipkart sales data to uncover key business insights such as:

* ✅ **Total Sales & Profit** trends
* ✅ **Top-performing Categories & Subcategories**
* ✅ **State-wise & City-wise Performance**
* ✅ **Customer & Order Insights**
* ✅ **Monthly and Quarterly Trends**

The dashboard is perfect for quick decision-making by stakeholders and business managers.

---

## 📂 Dataset

The dataset contains the following columns:

* **Order ID** 🆔 – Unique identifier of each order
* **Order Date** 📅 – When the order was placed
* **Customer Name** 👤 – Customer placing the order
* **State / City** 🌍 – Customer location
* **Category & Sub-Category** 🏷️ – Product details
* **Sales, Profit, Quantity** 💰 – Financial and sales metrics

---

## 🎨 Dashboard Highlights

✨ **Dynamic KPIs:** Total Sales, Profit, and Quantity Sold
✨ **Pivot Tables & Pivot Charts:** Summarizing and visualizing data effectively
✨ **Slicers & Timelines:** Interactive filtering for dates, states, and categories
✨ **Professional Design:** Clean layout, conditional formatting, and easy-to-read visuals

## 🚀 How to Use

1. Clone or download this repository.
2. Open `flipkart_sales_data_project.xlsx` in **Microsoft Excel**.
3. Use slicers and filters to explore the data.

```bash

 ├── 📄 flipkart_sales_data_project.xlsx  # Main Excel dashboard
```
